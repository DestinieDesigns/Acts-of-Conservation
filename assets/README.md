# Asset Map

Pixel art assets are organized by gameplay use.

- `character-tokens/`
  Selectable character and moving board token sprites.
- `grassland-tiles/`
  Grassland terrain and object sprites.
- `desert-tiles/`
  Desert terrain and object sprites.
- `volcanic-tiles/`
  Volcanic terrain and object sprites.

Each folder keeps the original `.tsj` tileset metadata next to the cropped PNGs used by the game.
